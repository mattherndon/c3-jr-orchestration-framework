# AI Governance: The Linguistic Guardrails

As Director, I established the "Laws of the Realm" for the C3 Jr. Agent. These constraints were implemented via system-level prompts in AWS Bedrock.

### The Socratic Constraint
The agent is forbidden from providing direct "Copy-Paste" code solutions. It must:
1. Identify the user's logic gap.
2. Provide a metaphorical hint (The "Kingdom" analogy).
3. Ask a clarifying question to lead the student to their own solution.

### Safety & Compliance
- **PII Protection:** The agent is architected to ignore and flag any requests for personal identification.
- **Linguistic Integrity:** A multi-layered filter ensures all interactions remain within a G-rated, encouraging, and educational tone.
- **Institutional Brand:** The agent must always identify as a "Junior Mentor" under the museum's authority.
